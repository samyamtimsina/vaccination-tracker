# Vaccination Tracker 💉

A modern web application for tracking child and mother vaccination records, built with React, Vite, and Prisma. Designed for ease of use, security, and accessibility for Nepali users.

## Features

- 👶 **Child & Mother Registration:** Easily add and manage profiles.
- 📅 **Vaccination Scheduling:** Track progress and upcoming vaccinations.
- 🔔 **Smart Alerts:** Automated reminders and notifications.
- 🛡️ **Secure Data:** Your health records are safe and private.
- 📊 **Reports & Analytics:** Generate and view vaccination statistics.
- 🌗 **Dark/Light Theme:** Switch between light and dark modes.
- 🇳🇵 **Nepali Language Support:** Localized UI for Nepali users.

## Tech Stack

- **Frontend:** React, Vite, DaisyUI, TailwindCSS
- **Backend:** Node.js, Express, Prisma
- **Database:** PostgreSQL (via Prisma ORM)
- **Authentication:** JWT-based secure login

## Getting Started

### Prerequisites

- Node.js & npm
- PostgreSQL

### Installation

```sh
git clone https://github.com/yourusername/vaccination-tracker.git
cd vaccination-tracker
```

#### Client

```sh
cd client
npm install
npm run dev
```

#### Server

```sh
cd server
npm install
npm run dev
```

### Environment Variables

- Configure your database and secrets in `server/.env`.

### Scripts

- `npm run dev` – Start development server
- `npm run build` – Build for production
